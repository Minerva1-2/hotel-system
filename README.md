基于gec6818的酒店房卡管理系统的设计与实现
项目内容：
1. 嵌入式终端功能：
2. （1）触摸屏登录功能：管理员通过账号密码登录系统
3. （2）客房状态显示功能：显示所有客房的当前状态
4. （3）入住登记功能：通过信息读取和RFID房卡绑定，完成客户入住登记
5. （4）续房办理功能：为在住客户办理续房
6. （5）退房结算功能：完成退房操作
7. （6）房卡管理功能：实现RFID房卡管理
8. （7）数据查询功能：支持客房信息查询、客户资料查询
9. 后台管理功能：
10. （1）系统用户管理：管理员账户的添加、删除
11. （2）数据备份管理：备份系统数据，支持数据恢复操作
     
  配置环境：
  上位机：GEC6818+QT5.7.0
  管理页面：Python+django
  因此，请在你的ubuntu中配置QT5.7.0交叉编译工具链并确保开发板可以正常运行QT代码
